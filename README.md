# Cyber Network Pathfinder

BFS, DFS, Dijkstra, A* pathfinding algorithms can be compared on the same animated grid map.

## Features

- Single algorithm visualization
- 2, 3, or 4 algorithm comparison mode
- Repeated average benchmark mode
- Custom grid size, wall count, and wall ratio
- Random or fixed start and goal points
- Optional weights and diagonal movement
- Time, energy, and step limits
- Execution history and saved history
- Korean and English language settings
- Dark and light themes
- Zoom controls for large maps

## How to Run

Open `index.html` in a browser.

No build step or server is required.

## GitHub Pages

1. Upload `index.html` and `README.md` to a GitHub repository.
2. Go to `Settings` -> `Pages`.
3. Set the source to `Deploy from a branch`.
4. Select the `main` branch and `/root`.
5. Open the generated GitHub Pages URL.

## Files

- `index.html`: main application
- `README.md`: project description

## Notes

This project stores preferences and run history in the browser's local storage.
